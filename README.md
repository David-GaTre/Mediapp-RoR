# MediApp v2.0
### Requisitos
* Tener Ruby instalado v3.0.4
* Tener postgresql instalado
* Tener yarn instalado

### Pasos para correr proyecto
```console
$ gem install bundler
$ bundle install
$ yarn install
$ rails db:create
$ rails db:migrate
$ rails server
```
El último comando abre la aplicación en http://127.0.0.1:3000
